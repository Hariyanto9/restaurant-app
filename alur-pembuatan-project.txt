alur pembuatan project katalog restoran untuk submission 2
1. membuat application shell, diawali dg membuat folder project dan intall npm dengan perintah "npm init --y"
2. memasang dependencies sebagai berikut
- @babel/core
- @babel/preset-env
- babel-loader
- copy-webpack-plugin
- css-loader
- eslint
- html-webpack-plugin
- style-loader
- webpack@4.x.x
- webpack-cli@3.x.x
- webpack-dev-server
- webpack-merge

dengan menggunakan perintah " npm install @babel/core @babel/preset-env babel-loader copy-webpack-plugin css-loader eslint html-webpack-plugin style-loader webpack@4.x.x webpack-cli@3.x.x webpack-dev-server webpack-merge --save-dev " 

perintah diatas akan memasang copy-webpack-plugin dan html-webpack-plugin versi terbaru:
Versi terbaru tersebut tidak kompatibel dengan webpack v4.x.x, sehingga akan memunculkan error saat menjalankan````npm run start-dev````.


TypeError: Cannot read property 'tap' of undefined

    at HtmlWebpackPlugin.apply



Gunakanlah copy-webpack-plugin versi 6 dan html-webpack-plugin versi 4.


npm uninstall html-webpack-plugin copy-webpack-plugin --save-dev 


npm install html-webpack-plugin@4.x.x copy-webpack-plugin@6.x.x --save-dev

tunggu hingga instalasi selesai, selanjutnya install regenerator run time dg perintah " npm install regenerator-runtime ", maka tampilah di package.json seperti dibawah ini: 

"devDependencies": {
    "@babel/core": "^7.10.3",
